# 💬 Java Multi-User Chat Application
This project is a robust **Client-Server Chat Application** developed in Java. It demonstrates core networking concepts including socket programming and multi-threaded communication.

## 🛠️ Technical Features
* **Socket Programming:** Real-time data transmission between server and multiple clients.
* **Multi-threading:** The server handles multiple client connections simultaneously using `ClientHandler`.
* **Object Serialization:** Efficiently sends message objects (Text, Images) across the network.
* **Swing GUI:** User-friendly interface for seamless interaction.

## 🚀 How to Run
1. Start the `ChatServer.java` first to listen for connections.
2. Run multiple instances of `ChatClientGUI.java` to start chatting.
